# Signal Parameters

<iframe src="main.html" width="100%" height="470px" scrolling="no" style="border:1px solid #ccc; border-radius:6px;"></iframe>

## How to Use

- **Amplitude slider** — set peak voltage (0.1–10 V).
- **Frequency slider** — sweep from 1 Hz to 1 kHz (logarithmic scale).
- **Phase slider** — shift the waveform −180° to +180°.
- **Show Reference** — overlay a zero-phase signal to visualize the phase offset.

## What to Observe

- The amplitude annotation marks the peak excursion from zero.
- The period bracket spans exactly one full cycle; period and frequency update in real time.
- Enabling the reference signal makes phase lead/lag visible as a horizontal shift.
- The info box shows \(f\), \(T\), \(\omega\), \(\varphi\), and \(V_{pp}\) simultaneously.

## Key Equations

\[v(t) = A\sin(\omega t + \varphi)\]

\[\omega = 2\pi f \quad,\quad T = \frac{1}{f}\]

\[V_{pp} = 2A\]
