## Description

This MicroSim helps students practice identifying nodes and branches in a circuit by highlighting each element on demand. Click any wire junction to see it labeled as a node, and click any component path to see it labeled as a branch. The sim reinforces the precise topological vocabulary needed before applying Kirchhoff's Laws.

## Key Concepts

- A **node** is any point in a circuit where two or more elements connect.
- A **branch** is any single two-terminal element or wire segment connecting two nodes.
- The number of branches and nodes determines the number of independent equations needed for analysis.
- Identifying topology correctly is the first step in nodal and mesh analysis.
