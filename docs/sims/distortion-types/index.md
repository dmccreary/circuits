## How to Use

- Select a distortion type using the buttons
- Adjust the drive level slider to increase or decrease the distortion amount
- Compare the input (clean) and output (distorted) waveforms

## What to Observe

- **Clipping**: flat tops appear when the amplifier saturates at the supply rails — harsh-sounding
- **Crossover distortion**: a notch appears at the zero crossing — characteristic of under-biased Class B stages
- **Harmonic distortion**: waveform shape changes smoothly — even-order sounds musical, odd-order sounds harsh
- **Intermodulation**: two input tones produce sum and difference frequencies unrelated to the originals
