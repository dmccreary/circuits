# Capacitor Combinations

## How to Use

- **C1, C2, C3 sliders** — set each capacitor value (1–100 µF).
- **C3 toggle** — add or remove the third capacitor to compare 2 vs 3 element combinations.
- Top panel shows the **series** configuration; bottom panel shows **parallel**.

## What to Observe

- Series equivalent is always **less than the smallest** individual capacitor.
- Parallel equivalent equals the **sum** of all capacitors.
- This is the **opposite** of resistors: resistors in series add directly, in parallel add reciprocally.

## Key Equations

**Series:**
\[\frac{1}{C_{eq}} = \frac{1}{C_1} + \frac{1}{C_2} + \frac{1}{C_3}\]

**Parallel:**
\[C_{eq} = C_1 + C_2 + C_3\]
